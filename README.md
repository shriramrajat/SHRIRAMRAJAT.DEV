## 🛑 The Thesis

Most engineers wait for a product manager to hand them a Jira ticket. **I refuse to be a ticket-taker.** 

As a student in Pune, my focus isn't on securing a standard 9-to-5—it is on architecting Micro-SaaS tools, optimizing backend pipelines, tracking live telemetries, and building systems with undeniable underlying taste. This repository holds the Next.js/Framer Motion frontend architecture representing my digital identity.

## ⚙️ Core Modules Built

This portfolio functions less like a gallery and more like a Command Center. It contains several meticulously designed architectural routes:

- **`/repoready` (Live Startup Pivot):** A deep-dive case study covering *RepoReady*, an automated pipeline that parses GitHub commits and transpiles them into actionable resume points using LLMs.
- **`/persona` (The operating Matrix):** A structural breakdown of my backend tooling, language constraints, and "The Graveyard" (a transparent log of my failed/scrapped systems to track real engineering growth).
- **`/work/sentinelstack` (Concurrency Case Study):** Documentation mapping an API Gateway I built to handle high concurrency using atomic Redis logic and decoupled Prometheus logging.
- **`/connect` (Active Terminal):** A fully instrumented live telemetry dashboard tracking local server times, role statuses, and secured channels to pull my Curriculum Vitae payload.

## 🛠️ The Tech Matrix (Frontend Architecture)

While my core strength is backend systems (FastAPI, Redis, PostgreSQL), this specific digital identity project is built on:

*   **Next.js 14+ (App Router):** Edge-optimized page routing and server-side logic.
*   **Framer Motion:** High-end, staggered animations and spatial UI states.
*   **Tailwind CSS:** Pure untethered styling mapped to a sleek, dark terminal aesthetic.
*   **React Hooks/State:** Tracking live telemetries and client-side interactions.

## 🚀 Local Initialization

If you want to spin up the local server to inspect the source code:

```bash
# Clone the repository
git clone https://github.com/shriramrajat/SHRIRAMRAJAT.DEV.git

# Initialize dependencies
npm install

# Boot development server
npm run dev
```

Intercept the port at `http://localhost:3000`.
